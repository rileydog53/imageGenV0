# Feedback Log

When a figure comes out wrong, low-quality, or surprising, log it here. This
file is the v2 priority list — the more concrete the entry, the more likely
the issue gets fixed.

Before logging, check [LIMITATIONS.md](LIMITATIONS.md): if the issue is an
already-known v1 limitation, no need to re-file it.

## How to log an entry

Copy the template, fill it in, add it under **Open** (newest first). Move it
to **Resolved** with a commit SHA when fixed.

```
### YYYY-MM-DD — short title
- **Prompt / IR:** what was asked for (prompt text or fixture path).
- **Expected:** what a correct figure should look like.
- **Got:** what was actually produced (attach or reference the PNG).
- **Severity:** blocker / major / minor / cosmetic.
- **Guess at cause:** optional — layout? primitive? style? verifier?
```
